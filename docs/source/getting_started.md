## What is vLLM Ascend?

vLLM Ascend plugin (vllm-ascend) is a community-maintained hardware plugin for running vLLM on the Ascend NPU. It provides a hardware-pluggable interface that decouples the integration of the Ascend NPU with vLLM, enabling popular open-source models to run seamlessly on Ascend hardware.
